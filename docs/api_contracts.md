# PresentOS API Contracts & Integrations

The system integrates with several third-party services to provide a seamless personal operating system experience.

## Integration Matrix

| Integration | Purpose | Details |
| :--- | :--- | :--- |
| **Notion** | Central DB | RPM hierarchy, tasks, XP, preferences. |
| **Fireflies** | Meetings | Transcribe/extract action items. |
| **Google Calendar/Gmail** | Sync | Two-way sync, auto-actions for scheduling and email. |
| **WHOOP/MD** | Biometrics | Energy-aware decision making. |
| **Weather/Surf APIs** | Environment | Surfline/OpenWeather for rescheduling activities. |
| **Monarch/Wealthfront** | Finance | Financial tracking and auto-investment tasks. |
| **Telegram/WhatsApp/SMS** | Access | Bots for chat, reminders, and proactive alerts. |
| **ElevenLabs/Whisper** | Voice | Natural Text-to-Speech (TTS) and Speech-to-Text (STT). |
